Cdo
==============================

Description
~~~~~~~~
CDO is a collection of command line Operators to manipulate and analyse Climate and NWP model Data.

Versions and Dependencies
~~~~~~~~
- 1.9.5
   #. fftw/3.3.7
   #. hdf5/1.8.16
   #. netcdf/4.5.0
   #. udunits2/2.2.24

Module
~~~~~~~~
You can load the modules by::

    module load cdo
