Mpc
==============================

Description
~~~~~~~~
Gnu Mpc is a C library for the arithmetic of complex numbers with arbitrarily high precision and correct rounding of the result.

Versions and Dependencies
~~~~~~~~
- 1.1.0
   #. gmp/6.1.2
   #. mpfr/3.1.6

Module
~~~~~~~~
You can load the modules by::

    module load mpc
