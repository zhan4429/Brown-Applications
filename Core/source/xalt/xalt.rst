Xalt
==============================

Description
~~~~~~~~
XALT is a lightweight software tool for any Linux cluster, workstation, or high-end supercomputer to track executable information and linkage of static shared and dynamically linked libraries. When the code is executed, wrappers intercept both GNU linker (ld) to capture linkage information and environmental variables.

Versions and Dependencies
~~~~~~~~
- 1.1.2

Module
~~~~~~~~
You can load the modules by::

    module load xalt
