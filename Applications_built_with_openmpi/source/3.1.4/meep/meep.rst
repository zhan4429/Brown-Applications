Meep
==============================

Description
~~~~~~~~
Meep or MEEP is a free finite-difference time-domain FDTD simulation software package developed at MIT to model electromagnetic systems.

Versions and Dependencies
~~~~~~~~
- 1.20.0
   #. fftw/3.3.7
   #. hdf5/1.8.16
   #. openmpi/3.1.4-old

Module
~~~~~~~~
You can load the modules by::

    module load meep
