Paraview
==============================

Description
~~~~~~~~
ParaView is an open-source, multi-platform data analysis and visualization application.

Versions and Dependencies
~~~~~~~~
- 5.9.1
   #. libtiff/4.0.10
   #. openmpi/3.1.4-old
   #. zlib/1.2.11

Module
~~~~~~~~
You can load the modules by::

    module load paraview
