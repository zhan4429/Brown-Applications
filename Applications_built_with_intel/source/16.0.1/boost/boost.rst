Boost
==============================

Description
~~~~~~~~
Boost provides free peer-reviewed portable C++ source libraries, emphasizing libraries that work well with the C++ Standard Library.

Versions and Dependencies
~~~~~~~~
- 1.64.0
   #. zlib/1.2.11

- 1.66.0
   #. zlib/1.2.11

Module
~~~~~~~~
You can load the modules by::

    module load boost
